# 💕 Just Us

**Just Us** is een frontend webapplicatie die ik heb gebouwd als persoonlijk leerproject tijdens mijn vakantie. Het doel van dit project was niet om een productieklare applicatie te ontwikkelen, maar om mijn vaardigheden met het Nuxt-ecosysteem verder uit te breiden en ervaring op te doen met het bouwen van interactieve webapplicaties.

Tijdens de ontwikkeling heb ik gewerkt met API's, dynamische filtering, formulieren, state management en een moderne UI.

---

## 🚀 Features

* 🔍 Zoek naar date-ideeën op titel.
* 💰 Filter op prijs.
* 📅 Filter op periode.
* 🏠 Filter op binnen- of buitenactiviteiten.
* 🎲 **Surprise Me**-functie die automatisch een willekeurige date kiest wanneer je zelf geen keuze kunt maken.
* ❤️ Sla favoriete dates op.
* ➕ Voeg zelf nieuwe date-ideeën toe.
* ✨ Scroll-animaties voor een vloeiende gebruikerservaring.
* 📱 Volledig responsive ontwerp.

---

## 🛠️ Tech Stack

* **Nuxt**
* **Nuxt UI**
* **JavaScript**
* **Tailwind CSS**
* **ScrollReveal**
* **JSON Server** (Mock REST API)

## 📡 Mock API

Voor dit project heb ik een eigen mock API opgezet met **JSON Server**. Hiermee kan de applicatie data ophalen en nieuwe date-ideeën toevoegen, waardoor de frontend werkt alsof deze verbonden is met een echte backend.

## ⚙️ Installatie

Clone de repository:

```bash
git clone <repository-url>
```

Installeer de dependencies:

```bash
npm install
```

Start de mock API:

```bash
npx json-server --watch db.json --port 3001
```

Start daarna de Nuxt development server:

```bash
npm run dev
```

De applicatie draait op:

```
http://localhost:3000
```

De mock API draait op:

```
http://localhost:3001
```
